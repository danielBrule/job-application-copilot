# Prompt resources

Prompts are version-controlled application resources. Their production wording and version
management will be implemented by the corresponding assessment and CV-generation tickets.

The required prompt slots are:

- `assessment/assessment.txt`
- `generation/english/stage_1.txt`
- `generation/english/stage_2.txt`
- `generation/english/stage_3.txt`
- `generation/english/stage_4.txt`
- `generation/french/stage_1.txt`
- `generation/french/stage_2.txt`

French generation always follows all four English stages.
